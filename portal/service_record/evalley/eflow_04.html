某プロバイダ・ルーターのJava認定通過

Tasks
Javaバージョンを謳うためにSun Micro社の認定に合格しないとなりませんでした。

利用していたJavaVMは Esmertec社(スイス企業) 設計の VMで多岐にわたる組込デバイスに移植出来るよう、移植環境まで提供された、大規模な「プラットフォーム」でした。

このプラットフォーム自体が認定に通すために設計されていました。なので、認定のための作業はほとんど無用でしたが、ルーター特有の機能をJavaのクラスから利用しようとしたコードのために対応が必要になりました。

Development Environment

  • Java ME
  • Esmertec社 Java VM開発プラットフォーム(Windows上：EclipsがUI/UXの巨大ソースコードファイルシステム)
